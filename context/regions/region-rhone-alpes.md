# Région 5 : Rhône-Alpes (Auvergne-Rhône-Alpes)

## Identité agronomique

- **Petite région agricole** : Plaine de la Limagne, Bresse, Dombes, Bas-Dauphiné, vallée du Rhône
- **Type de sol** : Très variable — Limagne (terres noires volcaniques exceptionnelles), Bresse (argiles), Dauphiné (alluvions)
- **Potentiel agronomique** : Très variable selon sous-régions (excellent en Limagne, moyen ailleurs)
- **Pluviométrie** : 600-900 mm/an (très variable selon altitude et exposition)
- **Irrigation** : Localement importante (vallée du Rhône, Dombes)
- **Altitude** : 150-600 m
- **SAU moyenne des exploitations** : 80-150 ha (très variable)

```
Année 1 : Colza
Année 2 : Blé tendre
Année 3 : Blé dur
Année 4 : Tournesol (ou Pois)
Retour Année 1
```

### Rotation type (Bresse / Dombes — avec maïs)
```
Année 1 : Colza
Année 2 : Blé tendre
Année 3 : Maïs grain
Année 4 : Blé tendre
Année 5 : Soja ou Pois
Retour Année 1
```

### Rotation type (Velay / Haute-Loire — altitude)
```
Année 1 : Lentille verte du Puy (AOP)
Année 2 : Blé tendre
Année 3 : Orge de printemps (ou Triticale)
Année 4 : Blé tendre
Retour Année 1
```

## Spécificités régionales

- **Limagne** : terres noires volcaniques parmi les plus fertiles de France, rendements blé comparables à la Beauce
- **Diversité** : grande variété de situations agronomiques selon altitude et sous-région
- **Lentille verte du Puy** : AOP très rémunératrice, créneau de diversification unique
- **Noix de Grenoble / Noix du Périgord** : arboriculture en complément (pas dans le périmètre grandes cultures)
- **Aides montagne** : ICHN pour les zones défavorisées (Haute-Loire, Cantal limitrophe)
- **TCS** : adoption du semis direct et techniques simplifiées en progression (sols argilo-calcaires)
- **Coopératives** : Limagrain (siège en Limagne), Axéréal, Oxyane
- **Risques** : gel tardif (altitude), sécheresse estivale, grêle (orages)
- **Enjeux** : adaptation montagne, valorisation des signes de qualité (AOP), diversification, irrigation raisonnée
- **Particularité Limagrain** : présence du groupe semencier qui propose des variétés testées localement
