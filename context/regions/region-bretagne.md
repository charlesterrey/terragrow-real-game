# Région 3 : Bretagne (Ouest)

## Identité agronomique

- **Petite région agricole** : Bassin de Rennes, Léon, Trégor, Cornouaille
- **Type de sol** : Limons sur schiste, sols légers acides, granite altéré
- **Potentiel agronomique** : Moyen à bon — historiquement polyculture-élevage, reconversion vers grandes cultures
- **Pluviométrie** : 700-1000 mm/an (excès hivernal fréquent, bonnes réserves)
- **Irrigation** : Rare (pluviométrie suffisante)
- **Altitude** : 50-250 m
- **SAU moyenne des exploitations** : 80-150 ha (en augmentation rapide)

### Rotation type (Bretagne céréalière)
```
Année 1 : Colza
Année 2 : Blé tendre
Année 3 : Maïs grain
Année 4 : Blé tendre
Année 5 : Pois ou Féverole
Retour Année 1
```

### Rotation type (ex-éleveur reconverti)
```
Année 1 : Colza (ou Légume industrie)
Année 2 : Blé tendre
Année 3 : Triticale
Année 4 : Maïs grain
Année 5 : Blé tendre
Retour Année 1
```

## Spécificités régionales

- **Reconversion** : nombreuses exploitations en transition de polyculture-élevage vers grandes cultures pures (cessation laitière)
- **Sols acides** : nécessité de chaulage régulier (amendements calcaires)
- **Pression mildiou** : climat océanique favorable aux maladies fongiques → pression fongicide plus forte
- **Légumes industrie** : opportunités de diversification avec Bonduelle, d'Aucy (contrats)
- **Matière organique** : disponibilité de fumier/lisier (élevage voisin) → économie d'engrais possible
- **CUMA** : très fort réseau de CUMA en Bretagne, mutualisation du matériel fréquente
- **Coopératives** : Le Gouessant, Triskalia (devenu Eureden), CECAB
- **Enjeux** : zones vulnérables nitrates (directive), captages d'eau potable, algues vertes
