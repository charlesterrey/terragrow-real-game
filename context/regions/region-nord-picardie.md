# Région 2 : Nord-Picardie (Hauts-de-France)

## Identité agronomique

- **Petite région agricole** : Picardie, Artois, Cambrésis, Vermandois, Santerre
- **Type de sol** : Limons battants (riches mais sensibles à la battance), argilo-limoneux
- **Potentiel agronomique** : Très élevé — première région céréalière de France
- **Pluviométrie** : 650-800 mm/an (bien répartie)
- **Irrigation** : Très peu (pluviométrie suffisante sauf années exceptionnelles)
- **Altitude** : 50-200 m
- **SAU moyenne des exploitations** : 120-200 ha

### Rotation type (Picardie betteravière)
```
Année 1 : Betterave sucrière
Année 2 : Blé tendre
Année 3 : Pomme de terre (ou Lin fibre)
Année 4 : Blé tendre
Année 5 : Colza (ou Pois)
Année 6 : Blé tendre
Retour Année 1
```

### Rotation type (Picardie sans pomme de terre)
```
Année 1 : Colza
Année 2 : Blé tendre
Année 3 : Betterave sucrière
Année 4 : Blé tendre
Année 5 : Orge d'hiver (ou Pois)
Retour Année 1
```

## Spécificités régionales

- **Betterave** : région historique, proximité des sucreries (Tereos, Cristal Union, Saint-Louis Sucre)
- **Pomme de terre** : contrats avec McCain, Bonduelle, marché frais — très rémunérateur mais capitalistique
- **Lin fibre** : première région française, coopératives de teillage (Terre de Lin, Calira)
- **Pluviométrie favorable** : meilleurs rendements céréaliers de France
- **Risques** : battance des sols (croûte de surface), jaunisse nanisante (betterave), pression adventices
- **Enjeux** : alternatives aux néonicotinoïdes (betterave), diversification des rotations, agriculture de conservation
- **Coopératives** : Noriap, Unéal, Tereos, Bonduelle
