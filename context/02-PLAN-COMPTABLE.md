# Plan Comptable Agricole Détaillé — The Real Game

## Principe

Ce plan comptable est adapté du Plan Comptable Général Agricole (PCGA), avec un niveau de détail à 3-4 chiffres permettant une ventilation par atelier/culture. Il correspond au niveau d'un cabinet de gestion type CER France / Cerfrance.

Les comptes sont organisés en 7 classes conformément au PCG.

## Ventilation analytique par culture

Pour le calcul des coûts de production, les comptes de charges (classe 6) sont ventilés analytiquement par culture à l'aide d'un code analytique :

| Code | Culture |
|------|---------|
| BT | Blé tendre |
| BD | Blé dur |
| OH | Orge d'hiver |
| OP | Orge de printemps |
| CZ | Colza |
| MG | Maïs grain |
| TO | Tournesol |
| PP | Pois protéagineux |
| FV | Féverole |
| BS | Betterave sucrière |
| PT | Pomme de terre |
| LF | Lin fibre |
| SJ | Soja |
| SG | Sorgho |
| TT | Triticale |
| LV | Lentille verte |
| GX | Charges générales (non affectées) |

### Règles de ventilation

1. **Charges directement affectables** (601, 6013, partie de 602) → affectées directement à la culture
2. **Charges de mécanisation** (amortissements matériel, fuel, entretien) → réparties selon le temps d'utilisation par culture ou au prorata des ha
3. **Charges de structure** (fermage, MSA, assurances, frais généraux) → réparties au prorata de la surface ou selon une clé spécifique
4. **Rémunération exploitant** → répartie au prorata de la surface ou du temps de travail

Cette ventilation analytique est la base du calcul des **coûts de production par culture**.
